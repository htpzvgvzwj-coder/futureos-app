# OCBC FutureOS

OCBC FutureOS is an AI-native life management layer embedded inside an OCBC-style mobile banking prototype. It explores how banking can move from managing financial products to managing customer life outcomes.

## FutureOS Phase 2 Product Bible

The Phase 2 Product Bible is the internal operating documentation system for future product, AI, design, banking, and engineering decisions.

- [English documentation](./docs/phase-2/en/00_README.md)
- [Simplified Chinese documentation](./docs/phase-2/zh-CN/00_阅读指南.md)

The documentation preserves the three-system architecture:

- Life Graph: customer understanding.
- Future Mirror: simulation and trade-offs.
- Future Self Guardian: monitoring, recommendations, approved action, and memory.

FutureOS should not add new top-level systems unless a Product Decision Record approves the change.
